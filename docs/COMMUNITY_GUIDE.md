# Community Guide

## Current collaboration rule

OpenDaoAgent is still in the founding stage.

Please prefer:

- small and reviewable changes
- scoped proposals
- practical architecture discussion
- phase-aware planning

Please avoid:

- turning distant vision into current implementation claims
- broad, unspecific requests without scope
- pushing DAO, token, or AGI narratives into Phase 1 work

## Best ways to join

1. Read `README.md`
2. Read `docs/FIRST_MVP.md`
3. Read open issues
4. Pick a small issue or open a scoped discussion
